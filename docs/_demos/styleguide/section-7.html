7.0 Environment

Add several top-level classes, that can change the color of all the env element below. This might be extremely useful when you have a dominant color for an application, or a composant. You can change it seemlessly by changing the top level env-<*> class. See (the env page on the wiki) by simply using the .btn-env class for additional informations.

  • .env-pool

    all the <*>-env element below are now of pool color.
