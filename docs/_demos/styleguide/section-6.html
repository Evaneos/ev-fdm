6.1 Checkboxes

You can style checkboxes using some of the following classes. However, take a close look to the associated html. You'll need to specify a <label> tag in order for this to work. Note that you could any icon specific class to that last tag for further personalisation (like color for instance)

  • .tick-checkbox

    A checkbox with a tick. Standard one.
  • .star-checkbox

    A checkbox star style. Useful for flagging a favorite.

     
<input id="{$modifiers}" type="checkbox" class="{$modifiers}">
<label for="{$modifiers}"></label>
